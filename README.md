# transaction-fraud-analytics-platform
End-to-end fraud analytics platform built on Google BigQuery (GCP)  using SQL, Python, and Power BI. Covers ETL pipeline development,  star schema data modeling, SQL based Analysis, and  interactive dashboard reporting, applied to real-world banking  transaction datasets to identify fraud patterns, customer risk  profiles, and operational trends.


## Datasets

Raw and cleaned datasets are not included in this repository
due to file size constraints.

**Source:**
IBM AML Transaction Dataset — available on Kaggle
https://www.kaggle.com/datasets/ealtman2019/ibm-transactions-for-anti-money-laundering-aml

**Storage:**
Cleaned datasets are loaded into Google BigQuery.
See gcp/bigquery_setup.md for setup instructions.
